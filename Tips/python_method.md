# Python Method 集

|   関数名   |                                                       処理内容                                                        |
| :--------: | :-------------------------------------------------------------------------------------------------------------------: |
|  `list()`  |                                              データ型をリストに変換する                                               |
| `split()`  |                               文字列をセパレータ文字列に基づき分割してリストに変換する                                |
| `append()` |                                                リスト末尾に要素を追加                                                 |
| `extend()` |                                          二つのリストを結合する (+=でも同様)                                          |
| `insert()` | リストのオフセット指定して、その前に要素を追加したいとき。オフセット 0 で先頭に、末尾を超えるオフセットで末尾に挿入。 |
| `remove()` |                                         削除したい値を指定して要素を削除する                                          |
|  `pop()`   |                                リストから要素を取り出して、取り出した要素を削除する。                                 |
|            |                                                                                                                       |
|            |                                                                                                                       |
