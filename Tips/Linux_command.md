# Linux command 備忘録集

### zip ファイル作成

`zip -r {~/zipファイル名}.zip {作成元フォルダパス}`

### zip ファイルにパスワードをかける

`zipcloak {zipファイルパス}`
`Enter password: {パスワード入力}`
`Verify password: {再度パスワード入力}`

### ディレクトリ内ファイル郡を圧縮, zip ファイル作成

`zip -er {~/zipファイル名}.zip {作成元フォルダパス}`
`Enter password: {パスワード入力}`
`Verify password: {再度パスワード入力}`
